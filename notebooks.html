Data Nominal/Categorical#

  • Nilai atribut nominal adalah simbol ataunama barang. Setiap nilai mewakili beberapa jenis kategori, kode, atau status, dan Atribut nominal juga disebut kategori. Nilai-nilainya tidak memiliki tingkatan nilai. Dalam ilmu komputer, nilainya juga dikenal sebagai enumerasi.

  • Melakukan pengukuran terhadap data nominal dengan menggukan rumus : $\( d(i,j)=\frac{p - m}p \)\( \) p \( = jumlah data yang akan dihitung jaraknya<br> \) m $ = jumlah data yang sama

# Menghitung jarak data Nominal/categorical
def d1(i,j):
    P = 4
    M = 0
    for col in ["Mobile_Phone", "Model", "Review","Payment_Method"]:
        if mobile[col][i] == mobile[col][j]:
            M += 1

    return (P-M)/P

# Menghitung jarak data Numeric dengan cara Euclidean Distance
def d2(i,j,h):
    result = 0
    for col in [hasil1, hasil2, hasil3, hasil4, hasil5]:
        result = math.pow(abs(col[i] - col[j]), h)
    return result**(1/h)

Equal Width#

Rumus perhitungan yang digunakan menggunakan Equal Width yaitu : $\( w=\frac{x_{min}-x_{max}}k \)$

\[v_i=x_{min}+i*w\]

\( x \) = nilai
\( i \) = 1,2,3,..
\( k \) = jumlah penglompokan
